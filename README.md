📖 Project Overview
This repository contains the official documentation and practical results of my first week as a Network Administrator Intern. The primary focus was on establishing a solid foundation in networking protocols, architectural models, and hands-on configuration using industry-standard tools.
🛠️ Core Tasks & Technical Knowledge
1. Conceptual Models (OSI & TCP/IP)
7-Layer OSI Model: Detailed analysis of all layers from Physical (L1) to Application (L7). I studied how each layer functions to ensure seamless data communication.
TCP/IP Suite: Comparison between the theoretical OSI model and the practical 4-layer TCP/IP model used in modern internet architecture.
Data Flow: Mastered the concepts of Encapsulation (adding headers/trailers) and Decapsulation (removing them) as data moves across the network.
2. Transport Layer Protocols
TCP (Transmission Control Protocol): Focused on connection-oriented, reliable delivery using the three-way handshake.
UDP (User Datagram Protocol): Explored connectionless, high-speed delivery used for streaming and gaming.
3. IP Addressing & Subnetting (The Core)
IPv4 Structure: Deep dive into 32-bit addressing, octets, and binary-to-decimal conversions.
Address Classes: Identification of Class A, B, and C ranges and understanding Private vs. Public IP boundaries.
Subnetting Mastery:
Performed manual calculations to divide networks into subnets.
Calculated Network Addresses, Broadcast Addresses, and Usable Host Ranges.
Implemented CIDR (Classless Inter-Domain Routing) to minimize IP wastage.
💻 Practical Implementations (Hands-on Labs)
Network Troubleshooting
Utilized Command Line tools like ping for connectivity, ipconfig/ifconfig for IP verification, and tracert/traceroute for path analysis on both Windows and Kali Linux.
Cisco Packet Tracer Labs
Initial Setup: Successful installation and environment setup of the Cisco simulation tool.
Switching: Built a simple topology connecting multiple PCs to a Layer 2 Switch, assigning static IPs, and verifying communication.
Router Configuration (CLI):
Accessing the Cisco IOS via Command Line Interface.
Configuring GigabitEthernet interfaces.
Assigning IP addresses and activating interfaces using the no shutdown command.
Topology Design
Evaluated the Star Topology for its reliability and scalability in LAN environments, understanding how a central hub/switch manages traffic.
🚀 Key Learning Outcomes
Ability to troubleshoot basic connectivity issues using the OSI framework.
Proficiency in manual subnetting for efficient network design.
Hands-on experience with Cisco IOS commands for device management
